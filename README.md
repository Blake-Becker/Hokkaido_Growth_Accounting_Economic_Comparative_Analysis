# Hokkaido_Growth_Accounting_Economic_Comparative_Analysis
Growth accounting analysis of Hokkaido and Tokyo using R-JIP 2017 regional productivity data to investigate why labor contribution declines more sharply in Hokkaido after 1992 and whether regional agricultural economic structure helps explain differences in long-run growth composition.
# Hokkaido Growth Accounting

This repository contains Python code and figures for a Japanese Economy project analyzing regional economic growth in Hokkaido relative to Tokyo.

The project uses a growth accounting framework to decompose output growth into:

- Capital contribution
- Labor contribution
- Productivity / residual contribution

The analysis focuses on how labor contribution changed before and after 1992 and how Hokkaido’s regional economic structure may help explain differences relative to Tokyo.

## Data

Source: Regional-Level Japan Industrial Productivity (R-JIP) Database 2017, RIETI.
